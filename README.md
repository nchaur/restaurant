## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nchaur/restaurant.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
     ```sh
   npm i bootstrap
   ```
   ```sh
   npm i reactstrap react react-dom
   ```
